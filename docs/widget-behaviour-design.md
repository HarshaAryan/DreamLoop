# DreamLoop Widget Behavior Design

The widget is the heart of the experience.

Users should feel like the world lives on their home screen.

---

# Core Widget Elements

The widget shows:

- characters
- environment
- weather
- current event
- time of day

The world should feel alive.

---

# Character Behavior

Characters should:

- walk around
- sit
- react to weather
- interact with environment

These actions create personality.

---

# Environmental Effects

The widget environment can change.

Weather effects:

Rain  
Snow  
Thunder  
Wind  
Falling leaves

Time effects:

Sunrise  
Day  
Sunset  
Night

These changes occur automatically.

---

# Mood Changes

The world mood can change.

Examples:

Cozy
Mysterious
Stormy
Magical

Visual cues should communicate mood.

---

# Event Display

When a story event occurs:

The widget highlights the event.

Users can open the decision prompt.

The event remains visible until both users choose.

---

# Idle Animation

Even when nothing happens:

The world should still move.

Examples:

Clouds drifting  
Leaves falling  
Fireflies glowing  

This keeps the widget alive.

---

# Rare Environmental Events

Occasionally the environment changes dramatically.

Examples:

Meteor shower  
Aurora sky  
Ghost fog  
Magic storm

These moments feel special.

---

# Widget Philosophy

The widget should feel like:

A tiny living world in your pocket.

Users should check it out of curiosity.
