# DreamLoop Feature Priority Roadmap

This document organizes features by development stage.

---

# Stage 1 — MVP

Goal:
Validate the core concept.

Features:

- user onboarding
- invite system
- character creation
- shared world
- story events
- decision choices
- scene updates
- widget display
- push notifications

Focus:

Simple and reliable interaction.

---

# Stage 2 — Engagement

Goal:
Increase daily usage.

Features:

- daily event schedule
- environmental changes
- weather effects
- time-of-day changes
- character animations
- memory system

These features make the world feel alive.

---

# Stage 3 — Emotional Depth

Goal:
Strengthen relationship experience.

Features:

- romantic moments
- cozy scenes
- bonding activities
- shared achievements
- story continuity

Users should feel the world remembers them.

---

# Stage 4 — Expansion

Goal:
Increase content variety.

Features:

- multiple world themes
- pets and creatures
- world decorations
- new story arcs

This stage increases replayability.

---

# Stage 5 — Monetization

Goal:
Generate sustainable revenue.

Possible offerings:

- premium worlds
- story packs
- cosmetic items
- special pets

These features should not harm the core experience.

---

# Stage 6 — Platform Expansion

Goal:
Extend ecosystem.

Platforms:

- Apple Watch
- Mac

Watch interactions should remain lightweight.

Mac experience can show expanded world view.

---

# Feature Philosophy

Each feature should answer one question:

"Does this increase emotional connection?"
