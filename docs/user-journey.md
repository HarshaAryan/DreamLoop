# DreamLoop User Journey

This document describes how users experience DreamLoop from discovery to long-term engagement.

---

# 1. Discovery

Users discover DreamLoop through:

- social media shares of story moments
- recommendations from friends
- app store discovery
- partner invitation links

The product should appear intriguing and emotional.

Users should immediately understand:

"A shared adventure world for two people."

---

# 2. First Install Experience

User downloads the app.

Initial onboarding should be short and magical.

Steps:

1. Welcome screen
2. Short animation introducing the dream world
3. Choose relationship type

Options:
- Couple
- Best Friend
- Family
- Adventure Partner

Tone of the story will adapt based on this choice.

---

# 3. Character Creation

Users create their character.

Customization includes:

- hairstyle
- clothing
- accessories
- color palette

Characters should feel cute and expressive.

Goal: emotional attachment.

---

# 4. Inviting a Partner

User invites someone.

Methods:

- link invite
- QR code
- contact invite

Once accepted:

The shared world is created.

---

# 5. World Creation

The first world is generated.

Examples:

- enchanted forest
- cozy village
- floating island

The world appears on the widget.

Characters appear together.

---

# 6. First Story Event

The first event appears.

Example:

"A glowing path appears in the forest."

Choices:

- Follow the path
- Ignore it
- Set up camp

Both players must choose.

---

# 7. Story Progression

After both players decide:

A new scene appears.

The world visually updates.

Example:

Characters move to a glowing cave.

---

# 8. Daily Engagement

Users receive events during the day.

Possible schedule:

Morning
Discovery events

Afternoon
Adventure events

Night
Dream events

These events create anticipation.

---

# 9. Emotional Moments

Occasionally events are peaceful.

Examples:

- sitting by a fire
- watching shooting stars
- feeding animals

These reinforce emotional bonding.

---

# 10. Rare Events

Rare events surprise users.

Examples:

- dragons
- secret portals
- haunted ruins

These moments become memorable.

---

# 11. Long-Term Engagement

Over time users unlock:

- new worlds
- special creatures
- decorations
- story arcs

The world evolves with them.

---

# 12. Sharing Moments

Special events generate shareable images.

Example:

"Episode 14: You discovered the dragon temple."

Users share these moments with friends.

This drives growth.

---

# 13. Long-Term Relationship Story

The story becomes unique to each pair.

No two worlds should be identical.

This creates emotional investment.
